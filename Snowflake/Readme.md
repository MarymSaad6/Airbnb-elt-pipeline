# ❄️ Snowflake Data Warehouse Project

## 📌 Overview
This folder contains all Snowflake-related scripts for building a complete data warehouse for the Airbnb dataset.  
The project follows a layered architecture (Raw → Staging → DWH) and includes schema design, data loading, and transformations.

---

## 🏗️ Project Structure
- **01_snowflake_setup/**
  - Row database, schema , Stages creation scripts.

- **02_DWH_Exploration/**
  - Exploration Datawarehouse and Final edits.
